## Clone jekyll theme
- 需要将项目clone下本地
下载主题
```sh
$ git clone https://github.com/Simpleyyt/jekyll-theme-next.git
$ cd jekyll-theme-next
```

安装Bundler
```sh
$ gem install bundler
```

安装依赖
```sh
$ bundle install
```
